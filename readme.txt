0. Разверните БД, данную в папке sql
0.1. Распакуйте файлы из папки с проектом
0.2. Внутри папки проекта найдите файл .env, откройте его и измените следующие строки:
	DB_HOST= ( ip сервера, если он есть )
	DB_PORT= ( порт сервера )
	DB_DATABASE= ( имя БД, для тестирования лучше оставить rpgu )
	DB_USERNAME= ( имя пользователя в БД )
	DB_PASSWORD= ( пароль пользователя в БД )

1. Перейдите на адрес проекта, вас должно перекинуть на страницу, где показываются все шаблоны
Текст письма шаблона на странице просмотра всех шаблонов представлен не полностью, для полного просмотра нужно перейти на страницу просмотра определённого шаблона.

Для создания нажмите на кнопку "Создать" вверху экрана.
Для просмотра шаблона нажмите на кнопку "Показать".
Для удаления шаблона нажмите на кнопку "Удалить".
Для редактирования шаблона сначала нажмите на кнопку "Показать", а затем на кнопку "Редактировать".

Серверная часть находится в папке "server"
В файле db.php находятся переменные для подключения к БД, которые нужно отредактировать
